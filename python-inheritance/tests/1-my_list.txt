>>> MyList = __import__('1-my_list').MyList

Instantiation works:

>>> ml = MyList()
>>> type(ml) == MyList
True

It inherits from list:

>>> isinstance(ml, list)
True
>>> issubclass(MyList, list)
True

Regular string representation:

>>> print(ml)
[]

Append works like a normal list:

>>> ml.append(1)
>>> ml.append(4)
>>> ml.append(2)
>>> print(ml)
[1, 4, 2]

print_sorted() with already sorted values:

>>> ml2 = MyList()
>>> ml2.append(1)
>>> ml2.append(2)
>>> ml2.append(3)
>>> print(ml2)
[1, 2, 3]
>>> ml2.print_sorted()
[1, 2, 3]

print_sorted() with unsorted values:

>>> ml3 = MyList()
>>> ml3.append(3)
>>> ml3.append(1)
>>> ml3.append(2)
>>> print(ml3)
[3, 1, 2]
>>> ml3.print_sorted()
[1, 2, 3]

print_sorted() with negative numbers:

>>> ml4 = MyList()
>>> ml4.append(5)
>>> ml4.append(-1)
>>> ml4.append(0)
>>> ml4.append(3)
>>> print(ml4)
[5, -1, 0, 3]
>>> ml4.print_sorted()
[-1, 0, 3, 5]

print_sorted() with empty list:

>>> ml5 = MyList()
>>> print(ml5)
[]
>>> ml5.print_sorted()
[]

print_sorted() must not modify the original list:

>>> ml6 = MyList()
>>> ml6.append(3)
>>> ml6.append(2)
>>> ml6.append(1)
>>> ml6.print_sorted()
[1, 2, 3]
>>> print(ml6)
[3, 2, 1]

print_sorted() returns None:

>>> result = ml6.print_sorted()
[1, 2, 3]
>>> print(result)
None
